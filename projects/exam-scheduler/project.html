Project Overview

The UChicago Exam Scheduler is a practical web application designed to solve a real problem faced by University of Chicago students: easily finding their final exam times and exporting them to their preferred calendar applications. The app provides a simple, intuitive interface for looking up exam schedules by class time or course code, and generates .ics files that can be imported into Google Calendar, Outlook, or any other calendar app.

Technical Implementation

This project was built using vanilla HTML, CSS, and JavaScript to ensure maximum compatibility and ease of deployment. The application uses a client-side approach with all exam data embedded in the JavaScript code, making it completely self-contained and requiring no server-side processing.

HTML5 CSS3 JavaScript (ES6+) iCalendar (.ics) Python (Data Processing)

Development Process

The project started with processing Excel data containing UChicago's final exam schedule. A Python script was developed to parse the Excel file and generate the necessary JavaScript mappings. The web interface was then built with a focus on simplicity and user experience, ensuring that students could quickly find their exam times without any learning curve.

Challenges & Solutions

Key Challenges Faced:

  • Data Processing: Converting complex Excel data into a usable JavaScript format
  • Calendar Format: Implementing proper iCalendar (.ics) file generation with correct timezone handling
  • User Experience: Creating an intuitive interface that works for all class time formats
  • Deployment: Making the application easily deployable without complex infrastructure

Future Enhancements

Potential improvements include adding support for different academic quarters, implementing exam conflict detection, integrating with UChicago's course catalog API, and adding mobile app functionality. The modular design makes it easy to extend the application with new features.
